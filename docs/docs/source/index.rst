dual_autodiff Documentation
===========================

Welcome to the `dual_autodiff` package documentation! This package provides tools for forward-mode automatic differentiation using dual numbers, ideal for educational and scientific computing purposes.

Overview
--------

The `dual_autodiff` package is designed to:

- Simplify the computation of derivatives for scalar functions.
- Provide a clean and intuitive API for differentiating mathematical functions.
- Ensure robust error handling and type checking for safe operations.
